# 国际象棋残局研究器

与上级目录的中国象棋版同架构：手工摆残局，白方（玩家）先行对黑方（AI）。

## 运行

```
python server.py          # 默认端口 8766
```

浏览器打开 `http://127.0.0.1:8766`（局域网地址启动时打印）。可与中国象棋版（8765）同时运行，页面顶部有互跳链接。

## 功能

与中国象棋版一致：摆盘校验、合法落点提示、求助最佳走法（绿色箭头）、
黑方可切换"本机 AI / 手动代走外部 AI"、悔棋 / 重试本局 / 重新布局、着法记录、
思考时间 1/4/16/64/256 秒、多核并行搜索、断线自动重连 + 任务续算 + localStorage 存档、
局面导出（含 **FEN**、局面代码、棋盘图、子力表）/ 导入（支持粘贴 FEN）。

## 规则说明

- 完整实现：将军/将死、**逼和=和棋**、吃过路兵、兵升变（**自动升后**）、
  三次重复判和（搜索树内同样感知）、子力不足判和（王对王、王单马/单象对王）。
- 不支持王车易位（残局摆盘场景用不到）；未实现 50 回合规则。
- AI：迭代加深 α-β + 静态搜索 + 置换表 + 空着裁剪（王兵残局自动关闭防 zugzwang）
  + LMR + 将军延伸 + 残局逼王入角评估（KQK/KRK 基本杀法可直接执行）。

## 测试基线

perft(1..3) = 20 / 400 / 8902 精确匹配；KQK 9 回合内将死；逼和、吃过路兵、升变均有用例。
