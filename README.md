# 中国象棋残局研究器

> **🌐 纯网页版（推荐）**：https://superddr.github.io/chess/ ——引擎已整体移植到
> JS Web Worker，浏览器本地计算（速度反超 Python 多进程版），无需任何安装，手机直接玩。
> 源码在 [web/](web/)，部署命令：`git subtree split --prefix=web -b t && git push -f origin t:gh-pages && git branch -D t`
>
> 以下为 Python 本地版说明（含对局监控日志等研究工具）。国际象棋版在
> [international/](international/) 子目录（端口 8766），功能一致，两个页面可互跳。

本地运行的中国象棋残局研究工具：手工摆残局，红方（玩家）对黑方（AI）。

## 运行

```
python server.py          # 默认端口 8000（若被系统保留可指定其它端口）
python server.py 8765     # 指定端口
```

无任何第三方依赖（仅 Python 标准库）。服务监听 `0.0.0.0`，启动时会打印两个地址：

- 本机访问：`http://127.0.0.1:8765`
- 局域网访问：`http://<电脑IP>:8765`（手机与电脑连同一 Wi-Fi 即可打开）

## 功能

- **摆残局**：点选棋盘下方棋子再点击棋盘放置；支持移除、清空、示例残局。摆放校验完整规则（九宫、象位、兵线、子数上限、开局不许照面/黑方被将）。
- **对弈**：红方先行，点击己方棋子会显示所有合法落点；AI 使用迭代加深 α-β 剪枝 + 静态搜索 + 置换表 + 将军延伸，多核 CPU 上自动以进程池并行搜索（核数-1 个 worker），思考时间可选 1/4/16(默认)/64/256 秒，搜到绝杀会提前返回。
- **黑方来源可切换**：默认本机 AI 应招；切到"手动代走"后可亲手替黑方落子（用于和外部 AI 对弈，把它的走法搬进来），随时可点「AI 代走黑棋」或切回自动。
- **避免重复局面**：AI 对重现历史局面的走法按次数扣分，不会无意义地来回挪子；长将则是硬禁着。
- **求助**：一键计算当前红方最佳走法，以绿色虚线箭头标出并给出中文着法（如"炮二平五"）。
- **规则完备**：蹩马腿、塞象眼、炮打隔子、王不照面、将死/困毙判定，**禁止长将**（同一将军局面反复出现即禁着，对玩家和 AI 双向生效）。
- **研究辅助**：悔棋、重试本局（回到摆好的初始残局）、重新布局、着法记录。
- **局面导出/导入**：一键把当前局面导出为文字（棋盘图 + 子力坐标 + 局面代码 + 着法记录），可直接粘贴给 AI 助手分析；也可粘贴局面代码导入复现。
- **断线恢复**：走棋/求助为"提交任务+轮询"模式，长考中断线服务器照常计算，重连后自动接上（同一请求重发命中同一任务，不重复计算）；对局状态实时存于浏览器 localStorage，刷新或重开页面自动恢复。

## 文件

- `engine.py` — 规则引擎与 AI 搜索
- `server.py` — 本地 HTTP 服务与 API（/api/start /api/moves /api/move /api/hint）
- `index.html` — 前端界面（Canvas 棋盘，自适应手机/电脑）
